# USAC AAC Website

## Tech Stack
- **Framework:** Next.js (React, App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Data Source:** Google Forms & Google Sheets
- **Backend Logic:** Next.js API Routes (serverless)

---

## Project Structure
- `app/`: Pages, layouts, and API routes
- `components/`: Reusable UI components (to be added)
- `lib/`: Data fetching / Google Sheets logic (to be added)
- `public/`: Static assets (images, icons, etc.)
- `README.md`: Project notes
- Config files: `tsconfig.json`, `next.config.ts`, `eslint.config.mjs`, `postcss.config.mjs`

---

## Local Development

Follow these steps to get the code onto your local environment:

```bash
git clone https://github.com/hillarynve/usac-aac-website
cd usac-aac-website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
